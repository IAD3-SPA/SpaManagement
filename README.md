## New Environment
python -m venv venv

## Activate Environment
venv\Script\activate

## Install library
pip install -r requirements.txt

## Django new project
django-admin startproject SpaManagement

## Django start app
cd SpaManagement
python manage.py startapp SpaApp

## Run Server
python manage.py runserver
